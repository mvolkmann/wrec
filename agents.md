# wrec

This is a library for building web components, similar to Lit and Stencil.

It is available in npm at https://www.npmjs.com/package/wrec.

The GitHub repository is at https://github.com/mvolkmann/wrec.
Many code examples are in the src/examples directory.

A blog page describing this is at
https://mvolkmann.github.io/blog/topics/#/blog/wrec/.
